## 一、為什麼要改

### 1.1 品恩的回饋（實際使用者測試）

品恩測試了三個念頭，反應一致：

| 念頭 | v2 改寫 | 品恩回應 |
|------|--------|--------|
| 「我不知道怎麼選擇才是對的」 | 「也許兩邊都有你在乎的東西」 | 「對，然後呢？」「像廢話」 |
| 「我覺得我一直在浪費時間」 | 「是不是心裡有個很想做的事，只是一直沒有動？」 | 「好像也不是」 |
| 「我永遠學不會安排時間」 | 「上一次覺得自己學不會某件事，後來是怎麼學會的？」 | 「有好一點點，但學某件事跟安排時間不一樣」 |

品恩的精確診斷（原話）：
- 「會有一種『對，然後呢？』的感覺」
- 「我好像很難被一句改寫就說服」
- 「我需要多一點對話和討論」
- 「應該是有一點後續可以進一步討論、讓我被說服的空間」
- 「改寫完念頭之後，我想回應的是：『對，兩邊都有我在乎的東西，可是我還是不知道怎麼選擇比較好』」

### 1.2 DTF 分析：問題的根源

品恩的回饋用 DTF 框架翻譯：

**v2 產生的預測誤差（PE）太小。**

改寫邏輯上正確，但落在品恩 g* 的低本徵值方向——她在那些方向不敏感，一句話就能 absorb 掉，v 幾乎沒動。

核心機制（來自 coaching_dtf_translation_dictionary.md §七.3）：
> 「認知拆解四問的 DTF 機制：每個問題在打開再鞏固窗口（回憶 → 激活 g）然後注入預測誤差（新問句 ≠ 原有預測）→ g 在窗口期被重塑。」

念頭改寫做的是這個鏈條的步驟 3：**在窗口已經打開的瞬間（使用者剛把念頭寫出來），注入一個夠大的 PE。**

v2 的 PE 不夠大 = g 沒被重塑 = 鞍點沒移動 = 「對，然後呢？」

### 1.3 v2 架構的結構性問題

v2 是三步：辨識 → 選品質目標 → 改寫

問題在 Step 2 和 Step 3 之間缺橋——選了「awareness」之後直接跳到寫句子，中間缺了「用什麼策略打中他」的決策。模型傾向給 generic 的角度轉換，而非針對這個人的精確命中。

## 二、改了什麼

### 2.1 架構從三步變四步

| | v2 | v3 |
|---|---|---|
| Step 1 | 辨識（渴望/擔心/假設 + 情緒引擎）| 分析狀況（+ 深度判斷 + 情緒溫度）|
| Step 2 | 選品質目標（insight/awareness/agency）| **2a 選品質目標 + 2b 選 PE 策略**（融合）|
| Step 3 | 七個方向工具箱 | 根據策略生成，大比例問句 |
| Step 4 | 品質自檢（連結/目標/語氣/開放性）| PE 檢查優先 + 連結 + 語氣 + 安全 |

### 2.2 核心改變：品質目標 × PE 策略的融合

品質目標 = 目的地（使用者感受到什麼）
PE 策略 = 交通工具（用什麼方式到達）
v2 只有目的地。v3 兩個都有。

自然組合矩陣：

| | 溯源 | 反例 | 校準 | 歸謬 | 換眼 | 體感 | 造新 | 遠觀 | 命名 | 並置 |
|---|---|---|---|---|---|---|---|---|---|---|
| **Insight** | ⭐ | | ⭐ | | | | | | | ⭐ |
| **Awareness** | | ⭐ | | ⭐ | ⭐ | ⭐ | | ⭐ | ⭐ | |
| **Agency** | | | ⭐ | | | | ⭐ | ⭐ | | |

### 2.3 新增 UX：「換一句」按鈕

策略性意義：
- 每次換用不同 PE 策略，系統性覆蓋
- 改寫可以更大膽（猜錯了使用者按換就好）
- 三句不同角度的改寫加起來 > 一句完美的改寫（對比本身產生 insight）

## 四、預期效果分析

### 4.1 預期改善

| 維度 | v2 實測 | v3 預測 | 原因 |
|------|--------|--------|------|
| PE 強度 | 低（品恩：「像廢話」）| 顯著上升 | 策略驅動取代 generic 角度 |
| 「對然後呢」機率 | 高 | 顯著下降 | 問句 + 策略精準度 |
| Insight | 4.40 | 4.6-4.8 | PE 策略更精準 |
| Awareness | 4.13 | 4.3-4.5 | 問句引導自我覺察 |

### 4.2 預期風險

| 風險 | 說明 | 緩解 |
|------|------|------|
| 語氣變銳利 | 高 PE 天然比較銳利 | Step 4 語氣檢查 |
| 問句疲勞 | 大比例問句像審問 | 情緒很熱時用陳述句接住 |
| 溯源法太深 | 輕量工具裡問「什麼時候開始的」可能嚇人 | 只在 identity 級語言時使用 |
| 歸謬法像辯論 | 「永遠？連一點可能都沒有？」| 軟化處理 + 限制條件 |
| 連結度波動 | 猜對很高猜錯很低 | 「換一句」按鈕兜底 |

### 4.3 品恩案例的 v3 預測改寫

| 念頭 | v2 | v3（融合架構）|
|------|-----|------------|
| 「不知道怎麼選」 | 「兩邊都有你在乎的」| 並置：「你比較怕選錯，還是怕選了回不了頭？」|
| 「一直在浪費時間」 | 「心裡有想做的事？」| 校準：「你說的浪費，是用誰的標準在衡量？」|
| 「永遠學不會安排時間」 | 「上次怎麼學會的？」| 校準：「你說的『很好的安排』，是什麼樣子？」|
